== Installation ==

Installing is easy:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Blendle Button screen to configure the plugin and save your credentials. No credentials yet? We can help you out!
4. Wrap the paid part of each blog post with the [blendlebutton] shortcode:
```
This is free content

[blendlebutton]
This is for paying visitors only.
[/blendlebutton]
```
5. Make sure the 'Enable Blendle Button' checkbox is checked (should be above the publish button)
6. Publish!

== Frequently Asked Questions ==

= What happens if I disable the Blendle button plugin? =

The paid part of your post will be rendered without the button and it will be visible to your visitors.
